DOM1 扩展

元素选择器

querySelect( )
根据 【css类名\id\元素名\元素属性】选择一个元素

querySelectAll( )
根据 【css类名\id\元素名\元素属性】选择元素 , 返回 NodeList 对象集合

👉 元素遍历

由于DOM1 在元素遍历时，IE9之前的浏览器不会返回文本节点，而其他浏览器会包含文本节点，导致格式不统一，所以DOM1扩展提供了一些属性，可以直接获取元素节点

children
返回所有子元素节点 (HTMLCollection 对象集合)

childElementCount
返回子元素节点的数量

firstElementChild
返回第一个子元素节点

lastElementChild
返回最后一个子元素节点

previousElementSibling
返回前一个同辈元素节点

nextElementSibling
返回后一个同辈元素节点

👉 classList 样式类操作

classList 返回了一个 DOMTolenList 集合对象，包含了元素的所有 class 类名

length 属性
返回类名的个数

item( index )
获取第 N 个类名

🗣 同样 DOMTolenList 也提供了一些方法来操作样式类

add( className )
添加类名 ， 如果已经存在则忽略

remove( className )
删除类名

toggle( className )
如果类名已存在则删除，如果不存在则添加

👉 style 样式类操作

元素节点拥有一个style对象，通过该对象提供的一些属性和方法可以控制元素在文档中的行内样式

cssText 属性
可以读取/设置(覆盖写入)完整的 style 样式代码

div.style.cssText = "width: 200px; heigjt: 200px"

getPropertyValye( styleName )
获取样式的值 , 如果查找的属性未设置则返回 "空字符"

setProperty( styleName , value )
追加行内样式

removeProperty( styleName )
删除一个行内样式

👉 应用在元素上的最终样式

window.getComputedStyle( tag )
返回一个 CSSStyleDeclaration 集合对象，包含了当前该元素在文档中最终生效的所有样式。
