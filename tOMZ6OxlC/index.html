Vue class 与 style 绑定

对象语法

<span :class="{active:  isActive}"></span>

是否应用 "active" 这个类名，取决与数据对象"isActive" 是否为true

对象语法2

<span :class="classObject"></span>

classObject: {
    active: true,
    isShow: true
}

通过一个对象一次控制多个类名
最终显示为

<span class="active isShow"></span>

数组语法

<span :class="[activeClass ,  isShowClass]"></span>

绑定的类名取数据对象 "activeClass" , "isShowClass" 的值

activeClass: "active",
isShowClass: "show"

最终显示为

<span class="active show"></span>

其他用法

三元表达式判断应用类名

<span  :class="[activeClass ? active : '']"></span>

数组语法中包含对象语法

<span :class="[ {active: }]"></span>
