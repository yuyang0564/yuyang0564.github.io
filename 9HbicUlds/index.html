switch 语句

switch 属于流程控制语句

switch (expression) { 
 case value: 
    statement 
 break; 
 case value: 
    statement 
 break; 
 default: statement 
}

👉 说明

  • switch 语句中的每一个 case 的含义是：“如果表达式等于这个值（value），则执行后面的
    语句（statement）”。
  • 通过 break 关键字跳出 switch 语句。
  • 如果省略 break 关键字，就会再执行完当前 case 后，继续执行下一个 case。
  • default 关键字则用于在表达式不匹配前面任何一个case时候，执行的默认代码

❗️ ❗️ 特别注意

case 不会发生隐式转换，它默认使用全等( === ) 操作符与 switch 的值进行比较
