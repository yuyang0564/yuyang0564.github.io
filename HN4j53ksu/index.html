css 垂直居中

通过 块元素 的 "before" 伪类实现垂直居中

  • 为父元素的伪类设置为 "inline-block" , 并设置 "vertical-align: middle"
.parent:before {
  content: "";
  display: inline-block;
  vertical-align: middle;
  height: 100%;
}
  • 子元素 同理
  • 因为 子元素 被转为了 inline-block , 所以相互之间会有一定的间隙
  • 设置 父元素 的 "font-size" 为 0 ，即可解决
.parent {
    ...
    font-size: 0;
}

通过 定位 垂直居中，同水平居中的定位方法(改变偏移的方向即可)

通过 flex 垂直居中

.parent {
    display: flex;
    align-items: center;
}
