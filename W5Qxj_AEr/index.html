Vue prop

子组件是通过 props 配置接收父组件传递的数据

export default {
    props: {
        list: Array
    }
}
  • 👉 除了 静态字符串 , 其他任意值都需要通过绑定属性的方式传递值
<child-component  v-bind:list="list"    />
<!-- 静态字符串 -->
<child-component str="abc"  />
<!-- boolean -->
<child-component  v-bind:bool="true" />
<!-- 数值 -->
<child-component  v-bind:num="10" />
  • 👉 如果想把一个对象的所有属性都传递给子组件 , 可以直接通过 v-bind 绑定一个对象
...
data = {
    id: 1,
    name: "li"
    ...
}

<child-component   v-bind="data" />

单向数据流

父子之间通过 prop 形成了一个单向下行绑定。
父级的数据更新时会自动同步到所有的子组件中，但是不能直接在子组件中直接修改 prop 数据。这样会导致其他用到此数据的组件也发生改变

非 prop 属性

当父组件传递数据给子组件 , 但是子组件没有定义 prop 来接收数据时， 父组件传递的数据会被添加到子组件的 根节点 上
👨 父组件

<child-component  v-bind:num="10"  />

👶 子组件未定义props 接收此数据时，会被渲染为

 <div num="10"> ... </div>
