Vue v-model

v-model 用来在表单类型中创建双向数据绑定

它会忽略表单元素 value 、 selected 、 checked 的初始值 ，而使用 v-model 绑定的数据作为初始值

<input  v-model="inputValue" />

export default {
    data() {
        return {
            inputValue: " "
        }
    }
}

v-model 是一个语法糖🍡，实际上它在底层监听用户的输入事件并更新数据

  • 针对 text 和 textarea 的 input，监听 input 事件， 同步 value 属性
  • 针对 checkbox 和 radio 的 input , 监听 change 事件 , 同步 checked 属性
  • 针对 select 元素 , 监听 change事件 ， 同步 value 属性

checkbox ☕️

<div>
        <label for="a">A</label>
        <input type="checkbox" id="a" value="A" v-model="checkboxList">
        <label for="b">B</label>
        <input type="checkbox" id="b" value="B" v-model="checkboxList">
        <label for="c">C</label>
        <input type="checkbox" id="c" value="C" v-model="checkboxList">
</div>

export default {
    return {
        checkboxList: [ ]
    }
}

radio ☕️

<div>
        <label for="a">A</label>
        <input type="radio" id="a" value="selectA" v-model="radio">

        <label for="b">B</label>
        <input type="radio" id="b" value="selectB" v-model="radio">

        <label for="c">C</label>
        <input type="radio" id="c" value="selectC" v-model="radio">
 </div>

export default {
    return {
        radio: " "
    }
}

textarea ☕️

<div>
        <textarea v-model="text"></textarea>
</div>

select ☕️

<div>
        <select v-model="select">
          <option value="" disabled selected>未选择</option>
          <option >选择一</option>
          <option >选择二</option>
          <option >选择三</option>
        </select>
</div>
  • 针对 select , 最好提供一个 disabled 的 option , 同时 value 为空，以提供默认值
  • 当没有为 option 提供 value 属性时， 会把 option 的值作为 value 的值
...
<option >选择一</option>
...
<!--  当选中时， value 的值为 " 选择一 " -->
