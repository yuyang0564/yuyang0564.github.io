3.2 innerText/outerText

innerText

1. 读取

获取该元素所有后代元素的文本内容

<ul>
    <li>
        a
        <div>
            b
        </div>
    </li>
</ul>

let ul = document.querySelect("ul")
ul.innerText
/*
    a
    b
*/

2. 写入

设置的内容将替换该元素的所有后代

<ul>
    <li>
        a
        <div>
            b
        </div>
    </li>
</ul>

let ul = document.querySelect("ul")
ul.innerText = "a"
/*
   <ul> a  </ul>
*/

注意: 如果设置的内容中存在 HTML 的语法字符，则会被转义 (包括 “小于号、大于号、引号及和号”)

<div>
  <p>子节点</p>
</div>

<script>
  ...
  div.innerText = "Hello & welcome, <b>\"reader\"!</b>";


  // 会被转义为
  /*
    <div>
      Hello &amp; welcome, &lt;b&gt;&quot;reader&quot;!&lt;/b&gt;
    </div>
  */
</script>

outerText

1. 读取

与 innerText 功能一致

2. 写入

写入时它会连同调用的元素节点一同替换
