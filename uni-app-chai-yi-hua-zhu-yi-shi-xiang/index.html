uni-app 差异化注意事项

🌈 1. 不要把 js,css,scss 等文件直接放入 static 文件中，因为此文件是不会进行编译的

🌈 2. 在css中和组件中引入样式，只能通过相对路径的方式

@import url("../../xx.css");

<style>
    @import "../../xxx.css"
</style>

🌈 3. 使用rpx 作为尺寸单位， 默认根据 750px 的宽度计算

🌈 4. 背景图片 不支持本地图片，可以转为base64 , 注意源文件不要大于 40Kb

🌈 5. 不支持 vue 的 classObject 的方式绑定样式

    <div :class="classObject"></div>
    <script>
        data: {
            classObject: {
                color: "#fff",
                fontSize: "16px"
            }
        }
    </script>

🌈 6. 不支持在自定义组件上使用 class 和 style 绑定

    <v-component :class="classStyle" />

🌈 7. 事件修饰符只支持 .stop 阻止事件冒泡 ， 其他修饰符只在 H5 端有效
