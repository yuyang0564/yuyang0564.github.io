字符串方法

📌 .includes( str, [index] ) 查找字符串中是否包含指定的字符

str : 指定的字符
index: 从索引位置开始查找
返回 true | false

let str = "hello word!"
str.includes("!")   //  true

📌 .startsWith( str, [index] ) 确认字符是否在字符串的开始位置

let str = "hello word!"
str.startsWith("!")   // false

📌 .endsWith( str, [index] ) 确认字符是否在字符串的结束位置

let str = "hello word!"
str.endsWith("!")   // true
str.endsWith("word!")   // true

📌 .repeat( num ) 字符串重复 n 次

  • 参数会自动取整 0 ~ -1 之间会取0 ， 但是 0 会返回 “ ”
  • x.1 ~ x.9 之间会取 x
  • 负数 和 Infinity 无效，会导致报错
  • 如果是字符串会使用隐式转换，如果不能正确转换为数值，则使用 0
let str = "a"
str.repeat(1)    // a
str.repeat('2')  //  aa 
str.repeat(undefined) // " "
str.repeat(null) // " "
str.repeat(NaN) // " "
str.repeat(1.9) // a
str.repeat(-5)  // " "
