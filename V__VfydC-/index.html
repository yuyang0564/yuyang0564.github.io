Vue slot

👉 slot 是组件的一块HTML模板，这块模板显示的内容由外界决定

👴 父组件中为子组件的slot 传值，传入的任何内容都会替换slot

<div id="app">
	<child-component>传入一段文本替换slot</child-component>
</div>
<script>
    ...
    data() {
        return {
            num: 1
        }
    },
    components: {
        listComponent
    }
</script>

👶 子组件中通过定义 slot 来接收外界传入的内容

<!-- export   child-component -->
<h1>
    <slot></slot>
</h1>

最终会渲染为 📽

👉 插槽也可以定义默认值，当外界没有传入内容时，会使用插槽的默认值

<!-- export   child-component -->
<h1>
    <slot>这是子组件插槽的默认值</slot>
</h1>

👉 在插槽内使用变量

使用规则: 父级模板里的所有内容都是在父级作用域中编译的；子模板里的所有内容都是在子作用域中编译的。

  • 在父级模板中替换插槽时，只能访问父级作用域
<div id="app">
	<child-component>
        {{ num  }}
    </child-component>
</div>

/*
...
data() {
    return {
        num: 0
    }
}
...
*/

👉 具名 slot

  • 当定义了多个 slot 时， 可以通过设置 name属性，为 slot 定义名称
<slot name="header"></slot>
<slot name="footer"></slot>
  • 不含有 name属性 的 slot ， 默认会带有一个隐含的 name 属性，值为 "default" （只能有 1 个）
<slot></slot>
<!-- 等价于 -->
<slot name="default"></slot>

👉 替换具名 slot

替换具名 slot ， 需要在 <template> 上使用 v-slot:name , 根据插槽的名称替换 name。
🗣 v-slot:name 简写 #name

<slot name="header">...</slot>

<!-- 替换 "header" 插槽 -->
<template v-slot:header>...</template>
<!-- 简写 -->
<template #header></template>

👉 作用域插槽

当在父组件中替换插槽时，只能访问父组件中的数据

// 父组件
<div>
    <template #default>
            <p>{{ num }}</p>
    </template>
</div>
/*
    ...
    data() {
        return {
            num: 0
        }
    }
    ...
*/

但是我们可以把子组件中的某个数据绑定到插槽上,让父组件使用。通过 v-bind:propname="prop"

// 子组件
<span>
    <slot :time="time"></slot>
</span>
/*
...
data() {
    return {
        time: '1990-01-01'
    }
}
...
*/

父组件访问插槽绑定的数据时, 通过 v-slot:name="propname" , 其中 propname 的名称可以自定义 ,

<div>
    <template v-slot:default="Obj">
        {{ Obj.time }}
    </template>
</div>

/*
    父组件的自定义 propname 相当于 
    propname = {
        time: 'xxx'
    }
*/

// 简写
<div>
    <template  #default="Obj">
        {{ Obj.time }}
    </template>
</div>

当然如果子组件绑定的数据是 map 类型，还可以解构出来

// 子组件
<span>
    <slot :user="user"></slot>
</span>
/*
...
data() {
    return {
        user: {
            name: "xx",
            age: 10
        }
    }
}
...
*/
/*---------------------------------分隔线-------------------------------*/

// 父组件
<div>
    <template  #default="{user}">
        {{ user.name }}
        {{ user.age }}
    </template>
</div>
