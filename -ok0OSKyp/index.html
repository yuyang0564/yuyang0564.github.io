DOM

3.1 innerHTML/outerHTML

innerHTML

1. 读取

读取innerHTML时,返回该元素的所有后代节点的字符串形式

<ul>
  <li>1</li>
  <li>2</li>
  <li>3</li>
</ul>
<script>
  const ul = document.quertSelect("ul")
  console.log(ul.innerHTML)

  /*
    <li>1</li><li>2</li><li>3</li>
  */
</script>

2. 写入

通过 innerHTML 写入时，将根据写入的字符串以新的DOM子数替换掉源子节点

<ul>
  <li>1</li>
  <li>2</li>
  <li>3</li>
</ul>
<script>
  const ul = document.quertSelect("ul")
  ul.innerHTML = "<li>替换的节点</li>"

  /*
    <ul>
      <li>替换的节点</li>
    </ul>
  */
</script>

如果写入的内容不含 HTML 标签则会生成一个文本节点

<div>
  <p>源节点</p>
</div>
<script>
  const div = document.quertSelect("div")
  div.innerHTML = "替换"

  /*
    <div>替换</div>
  */
</script>

outerHTML

1. 读取

读取时会连同调用的元素及其子元素一起,以HTML字符串的方式返回

<ul>
  <li>1</li>
  <li>2</li>
  <li>3</li>
</ul>
<script>
  const ul = document.quertSelect("ul")
  console.log(ul.outerHTML)

  /*
    <ul>
      <li>1</li>
      <li>2</li>
      <li>3</li>
    </ul>
  */
</script>

2. 写入

写入时，写入的内容将完全替换当前的元素

<div><p>1</p></div>
<script>
  const div = document.quertSelect("div")
  div.outerHTML = "<h1>替换</h1>"
</script>

以上面的代码为例,outerHTML相当于新建一个<h1>元素，然后替换掉 <div>

const p = document.createElement("h1")
h1.appendChild(document.createTextNode("替换"))
div.parentNode.replaceChild(p,div)
