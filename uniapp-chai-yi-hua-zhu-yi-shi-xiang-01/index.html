uniapp 差异化注意事项

🌈 static 文件内不要放 js 、 css 、 scss 等文件，因为它们不会经过编译

🌈 css 引入时只能使用相对路径

@import url ("../../common/base.css");

🌈 使用 process.evn.NODE_ENV 判断当前的开发环境, 打包时自动使用 production , 本地运行时自动使用 development

🌈 背景图片 小程序端不支持本地图片，需要转为base64 。 App端在v3模式下支持本地图片，但不要大于40kb
