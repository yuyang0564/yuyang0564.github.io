scss 运算符 " / "

/ 运算符通常情况下会被识别为 css 中的 “分隔”, 所以在被当作 “除法”运算符时，需要使用 ( ) 进行包裹

.box {
 width: (10 / 2) + px;
}

.box {
 width: (10px / 2);
}

$width: 10px;
.box {
 width: ($width / 2);
}

❗️❗️ 注意： 如果两个带有单位的数值直接相除时， 得到的值 "没有单位"

.box {
  width: (10px / 2px);
}

转换为 css

.box {
  width: 5;
}
