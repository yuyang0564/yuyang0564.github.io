Vue 指令

Vue 指令以 v- 开头 ，它是一种特殊的属性，它的值为一个表达式

<span v-if="isTrue"></span>

v-bind: 指令，用来动态的绑定属性

<span v-bind:href="url"></span>

绑定一个 href 属性，它的值为 url 数据对象的值

动态参数

<span v-bind:[attribute]="url"></span>

针对attribute 求值后，作为属性使用
注意： 不能使用 空格 和 引号

<span v-bind:[attribute + 'xxx' ]="url"> 错误示例 </span>

v-bind 简写

<span :href="url"></span>

<span :[attribute]="url"></span>

v-on: 指令 , 用来绑定事件

<button v-on:click="handleClick"></button>

为当前 button 绑定一个 click 事件， 事件处理方法为 " handleClick"

动态绑定事件

<button v-on:[eventHandle]="handleClick"></button>

针对 eventHandle 求值后，作为事件名

v-on 简写

<span @click="handleClick"></span>

<span @[attribute]="url"></span>
