类型检测

👉 typeof 主要用来检测一个值是 基本类型 还是 引用类型

类型 检测
String "string"
Number "number"
Boolean "boolean"
undefined "undefined"

除了以上会返回对应的基本类型 表示的字符串，其余均返回 "object"

📌 特殊： 对函数使用 typeof 会返回 "function"

👉 instanceof 确定对象的类型

由于所有引用类型 都是 Object 类型的实例，所以在检测所有"引用类型" 是不是 "Object" 类型的实例时，始终返回 true

[ ] instanceof Object

{ } instanceof Object

检测数组类型

[] instanceof Array

检测正则类型

\\ instanceof RegExp
